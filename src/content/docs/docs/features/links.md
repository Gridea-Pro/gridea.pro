友情链接（Friend Links）是博客圈互相推荐、建立联系的传统功能。Gridea Pro 提供了可视化的友链管理界面。

## 添加友链

1. 在左侧导航中点击 **友链** 进入管理页面
2. 点击 **新建链接** 按钮
3. 填写链接信息：
   - **名称** — 友站的名称
   - **URL** — 友站地址
   - **头像** — 友站的头像或 Logo（支持外部 URL 或上传本地图片）
   - **描述** — 对友站的简要介绍

4. 点击保存

## 编辑与删除

- 点击友链卡片可编辑所有信息
- 点击删除按钮移除友链
- 支持批量管理和排序

## 主题展示

友链数据在渲染时以 `links` 变量传递到主题模板。每个友链对象包含：

```
id          — 唯一标识
name        — 友站名称
url         — 友站地址
avatar      — 头像 URL
description — 描述文字
```

:::note
友链的展示效果取决于主题的实现。部分主题可能将友链渲染为独立页面，部分可能渲染为侧边栏组件。如果你的主题没有友链页面，你可能需要在主题模板中自行添加。
:::

## 数据存储

友链数据保存在站点目录下的 `config/links.json` 文件中。
