Gridea Pro 内置了 AI 助手功能，可以在写作过程中提供智能辅助。支持内置免费模型和自定义接入多种 AI 服务。

## 使用模式

### 内置免费模型

开箱即用，无需配置 API Key：

- 每日免费额度：20 次调用
- 每分钟限制：5 次调用
- 无需注册或付费

:::tip
内置免费模型适合轻度使用。如果你需要更频繁地使用 AI 功能，建议配置自定义模型。
:::

### 自定义模型

使用你自己的 API Key 接入 AI 服务，无调用次数限制。

**支持的 AI 服务商（13+）：**

| 服务商 | 代表模型 | 说明 |
|--------|----------|------|
| OpenAI | GPT-4、GPT-3.5 | 国际主流 |
| Anthropic | Claude | 长文写作优秀 |
| DeepSeek | DeepSeek | 国产高性价比 |
| Google | Gemini | 多模态能力 |
| Kimi（月之暗面） | Moonshot | 国内长文本 |
| 通义千问 | Qwen | 阿里云 |
| 智谱 | GLM | 国内 |
| Cohere | Command | 国际 |
| Ollama | LLaMA 等 | 本地部署 |
| 自定义 | — | 任意 OpenAI 兼容接口 |

## 配置 AI 助手

1. 在 **设置** 中找到 AI 配置区域
2. 选择使用模式（内置 / 自定义）
3. 如选择自定义：
   - 选择 AI 服务商
   - 填入 API Key
   - 选择具体模型
   - 点击 **测试连接** 验证配置
4. 保存设置

:::note
API Key 等敏感信息存储在系统钥匙串（Keychain）中，不会出现在配置文件里。
:::

## AI 功能

### 智能生成 Slug

在文章编辑器中，AI 可以根据中文标题自动生成 SEO 友好的英文 Slug。例如：

- 「深入理解 Go 并发编程」→ `understanding-go-concurrency`
- 「我的 2024 年终总结」→ `2024-year-in-review`

### 标题建议

基于文章内容，AI 可以推荐多个候选标题供你选择。

### 内容摘要

自动提取文章核心内容，生成简洁的摘要文字，可用于文章列表页展示。

### 写作辅助

在编辑器中获取语法修正、表达优化等建议。

### 标签推荐

根据文章内容自动推荐合适的标签。

## 工作流 Prompt

Gridea Pro 内置了 5 个预设工作流，覆盖常见的博客写作场景：

### 1. 写作助手

提供创意写作帮助，包括：
- 内容构思和大纲生成
- 段落扩展和润色
- 表达方式优化建议

### 2. 闪念成文

将多条 [闪念笔记](/docs/features/memos) 整理为一篇完整的博客文章：
- 自动归纳闪念主题
- 生成文章结构
- 补充过渡和衔接

### 3. 内容审核

对已写好的文章进行全面审查：
- 语法和拼写检查
- 逻辑一致性验证
- 可读性评估
- 改进建议

### 4. 站点健康检查

分析你的博客整体质量：
- 内容覆盖度评估
- 发布频率分析
- SEO 优化建议
- 改进方向推荐

### 5. 文章翻译

将文章翻译为其他语言：
- 保持 Markdown 格式
- 适当本地化表达
- 支持多语言目标

## 代理设置

如果你的网络需要代理才能访问 AI 服务，可以在 **设置 > 网络** 中配置代理。代理设置会应用于所有 AI 请求。

## 用量追踪

使用内置免费模型时，Gridea Pro 会追踪你的调用次数：
- 每日额度在次日自动重置
- 达到限额时会收到明确的提示信息
- 自定义模型不受此限制
