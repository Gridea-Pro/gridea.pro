Gridea Pro 不仅仅是一个静态博客编辑器，它还集成了丰富的内容管理和网站增强功能。本章节将逐一介绍这些核心功能的配置与使用方法。

## 功能概览

| 功能 | 说明 |
|------|------|
| [闪念笔记](/docs/features/memos) | 快速捕捉灵感，支持标签、图片和贡献热力图 |
| [标签与分类](/docs/features/tags-categories) | 多维度内容组织，支持彩色标签和层级分类 |
| [评论系统](/docs/features/comments) | 集成 7 大评论平台，一键开启读者互动 |
| [友情链接](/docs/features/links) | 管理友链列表，支持头像和描述 |
| [AI 助手](/docs/features/ai-assistant) | 内置 13+ AI 模型，辅助写作、翻译、摘要生成 |
| [SEO 设置](/docs/features/seo) | Sitemap、RSS、Open Graph、统计分析一站式配置 |
| [PWA 配置](/docs/features/pwa) | 让博客支持离线访问和桌面安装 |
| [CDN 与图片管理](/docs/features/cdn-media) | 图片 CDN 加速，支持又拍云、Cloudinary 等 |

## 推荐阅读顺序

如果你刚开始使用 Gridea Pro，建议按以下顺序了解各功能：

1. 先完成 [快速开始](/docs/getting-started) 中的基本配置
2. 学习 [标签与分类](/docs/features/tags-categories) 来组织内容
3. 尝试 [闪念笔记](/docs/features/memos) 记录碎片化想法
4. 开启 [评论系统](/docs/features/comments) 与读者互动
5. 配置 [SEO 设置](/docs/features/seo) 提升搜索引擎可见性
6. 按需启用 [AI 助手](/docs/features/ai-assistant)、[PWA](/docs/features/pwa)、[CDN](/docs/features/cdn-media)
